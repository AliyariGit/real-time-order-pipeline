# Real-Time Order Pipeline (Kafka Project)

This project demonstrates a simple real-time data pipeline using Kafka.

## 🧩 Architecture

Producer → Kafka → Consumer → Storage

## 🚀 Features

- Real-time order generation
- Event streaming using Kafka
- Data processing via consumer
- Storage simulation using file system

## 🛠️ Tech Stack

- Node.js
- Kafka (Docker)
- KafkaJS
